# Inventory Management System

A full-stack inventory management application built with **FastAPI**, **React**, and **PostgreSQL** to streamline warehouse operations, inventory tracking, and stock management. The system provides secure authentication, real-time inventory monitoring, and comprehensive reporting through a modern, responsive interface.

## Key Features

* JWT-based user authentication and role-based access control
* Product, category, and warehouse management
* Inventory operations including receipts, deliveries, transfers, and stock adjustments
* Real-time inventory tracking with low-stock alerts
* Dashboard with inventory analytics and reports
* Complete inventory movement history and audit logs
* Multi-warehouse inventory management
* RESTful API architecture with secure backend services

## Architecture

* RESTful API using FastAPI
* PostgreSQL database with SQLAlchemy ORM
* JWT-based authentication and authorization
* Modular backend architecture with service-based design
* Responsive React frontend with reusable components
